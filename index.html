Personal Webpage

This webpage showcases effects such as parallax as well as animations to create a responsive webpage to serve as a personal portfolio / website.
Key Features:

  • Dynamic title animations
  • smooth parallax scrolling
  • Stylized sections
  • Interactive buttons for easy navigation

Technologies utilized:
  • HTML, CSS for structure and design
  • Javascript for interactive elements / animations
  • Intersection Observer API for triggering animations while scrolling
  • Bootstrap framework for responsiveness and screen size adaptation
  • GitHub utilization


iOS App

This iOS app is a prototype for creating and manageing events that aids in managing client and event inforamtion, inspired by my work in the entertainment industry.
Key Features:

  • Event creation / management: Easily create new events with essential details such as name, location, date, and additional notes.
  • Profile tracking: Enables users to maintain client profiles by saving data such as name, contact details, email, and other data.

Technologies utilized:
  • SwiftUI Implementation: Allowing a smooth and dynamic user experience
  • Data storage: User-generated data is stored using JSON encoding and decoding.
  • Edit functionality: Ability to modify and delete details of a prevoiusly stored event, and saving the progress.
  • SwiftUI Navigation: utilizng navigation structures and vertical stacks to create seamless views transitions.
